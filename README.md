# Projeto Next.js

Projeto desenvolvido para prática de desenvolvimento web utilizando Next.js e TypeScript.

## Tecnologias utilizadas

* Next.js
* TypeScript
* HTML/CSS

## Funcionalidades atuais

* Estruturação de páginas
* Componentização
* Navegação entre páginas
* Interface responsiva

## Próximos passos

* Integração com banco de dados
* Sistema de autenticação
* Persistência de dados

## Objetivo

Aprimorar conhecimentos em desenvolvimento web moderno e arquitetura de aplicações front-end.
